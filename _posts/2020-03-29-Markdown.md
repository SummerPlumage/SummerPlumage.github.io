## 표

HYPHEN(-)갯수는 상관없다

```
|항목|가격|개수|
|:---:|:----:|:----|
|마스크|5000원|10개|
|너무|500원|20개|
|비싸|1000원|20개|
```

|항목|가격|개수|
|:---:|:----:|:----|
|마스크|5000원|10개|
|너무|500원|20개|
|비싸|1000원|20개|  

### 표 정렬

HYPHEN 좌우에 COLON(:) 붙여서 표현, 안쓰면 왼쪽정렬

```
|중앙정렬|오른쪽정렬|왼쪽정렬|
|:------:|-------:|:-------|
|마스크너무비싸|5000|112321310|
|너무|500|20|
|비싸|1000|20|
```

|중앙정렬|오른쪽정렬|왼쪽정렬|
|:------:|-------:|:-------|
|마스크너무비싸|5000|112321310|
|너무|500|20|
|비싸|1000|20|
